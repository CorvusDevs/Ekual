# Ekual

**Loudness Equalization for macOS**

Ekual normalizes audio volume across all your apps in real-time. Quiet dialogue and loud explosions are brought to the same comfortable level — no more reaching for the volume knob.

- Real-time loudness equalization for all system audio
- Lightweight menu bar app
- Customizable presets (Light, Medium, Heavy, or create your own)
- Exclude specific apps from processing
- Global keyboard shortcut to toggle on/off
- Runs at launch with near-zero CPU usage
